== Description ==
Title research enables you to find more informations about the title of your wordpress post from the posts's page directly.

For details about this plugin. Please go to http://www.shahariaazam.com/title-research-plugin-for-wordpress/


To keep you updated with the author's regular plugin development. Please go to http://www.shahariaazam.com

== Installation ==
Upload the 'Title Research' plugin to your blog, Activate it.
You're done!
